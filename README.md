# **ZhangGa**
A microblogging app. A user can create posts/articles and receive comments.

### Stack:
* ReactJS
* PostgresQL
* NodeJS

`master` branch uses Redux-thunk, to handle API calls and manage state

`allInState` branch only uses state to save data. Will not save data after refresh.

This project was bootstrapped with [Create React App](https://github.com/facebook/create-react-app).
